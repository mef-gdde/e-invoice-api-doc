# cbc:Party
The accounting customer party.

## About

* Occurrence: [1..1]
* Namespace: urn:oasis:names:specification:ubl:schema:xsd:CommonAggregateComponents-2


## Child Elements

| Data Field                                             | Occurrence | Definition                                                                                |
| ------------------------------------------------------ | :--------: | ----------------------------------------------------------------------------------------- |
| [cbc:EndpointID](./party/endpoint-id.mdx)              |   [1..1]   | The main identifier of this entity (CamInvoice ID). (e.g. KHUID00001234). |
| [cac:PartyName](./party/party-name.mdx)                |   [០..1]   | A name for this party. In our case, the business's khmer name.                            |
| [cac:PostalAddress](./party/postal-address.mdx)        |   [1..1]   | The party's postal address.                                                               |
| [cac:PartyTaxScheme](./party/party-tax-scheme.mdx)     |   [1..1]   | A tax scheme applying to this party.                                                      |
| [cac:PartyLegalEntity](./party/party-legal-entity.mdx) |   [1..1]   | A description of this party as a legal entity.                                            |
| [cac:Contact](./party/contact.mdx)                     |   [1..1]   | The primary contact for this party.                                                       |
