# cbc:InvoiceTypeCode
A code indicating the type of the invoice. Currently, there are 4 types of InvoiceTypeCode:
- 380 - Commercial Invoice
- 388 - Tax Invoice

## About
* Occurence: [0..1]
* Namespace: urn:oasis:names:specification:ubl:schema:xsd:CommonBasicComponents-2

## Attributes

## Example 
```xml
<cbc:InvoiceTypeCode>388</cbc:InvoiceTypeCode>
```
