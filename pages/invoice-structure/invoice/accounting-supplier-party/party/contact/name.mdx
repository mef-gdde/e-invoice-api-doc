# cbc:Name

A name for the entity.

## About

- Occurrence: [0..1]
- Namespace: urn:oasis:names:specification:ubl:schema:xsd:CommonBasicComponents-2

## Attributes

| Name             | Occurrence | Type               | Description                                    |
|------------------|:----------:|--------------------|------------------------------------------------|
| languageID       |   [0..1]   | xsd:language       | The identifier of the language used in the content component. |

## Example

```xml
