# cbc:URI
Represents the Uniform Resource Identifier (URI) that identifies an external object, typically as an Internet resource.

## About
* Occurrence: [1..1]
* Namespace: urn:oasis:names:specification:ubl:schema:xsd:CommonBasicComponents-2

## Example

```xml
<cbc:URI>http://www.suppliersite.eu/sheet001.html</cbc:URI>
```
