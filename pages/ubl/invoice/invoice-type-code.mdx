# cbc:InvoiceTypeCode
A code indicating the type of the invoice. As tax invoices are currently the only type in use, this field is optional.
## About
* Occurence: [0..1]
* Namespace: urn:oasis:names:specification:ubl:schema:xsd:CommonBasicComponents-2
